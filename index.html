SOCIAL NETWORKING

We are living in the age of Social Networking like Facebook, Linkdln, Google etc. In this project our Objective is to develop a social networking application. Functionalities which we include in our project are listed below:

  • User can register into the application with their email and password. Only one user per email id is allowed.

  • Registered user can login into application by providing appropriate credentials.

  • There is a seperate section where user can get basic information like date of birth, address, phone number, education, hobby etc.

  • Registered user can add new friends.

  • User can see profile information of their friends.

  • User can send text messages to their friend and recieve text from them.

  • User can check if his/her friend is active or not.

  • Html
